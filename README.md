## 트러블 슈팅

- **3D 프린팅 출력물 규격 불일치(조립 공차 문제)**
  - 문제: 3D 모델링한 부품이 조립 시 규격이 맞지 않거나, 출력물이 예상 치수/형상과 다르게 출력되는 문제가 발생했습니다.
  - 원인: 모델 스케일/단위 설정 오류, 프린터 캘리브레이션 미흡, 슬라이서 파라미터(수축·유량·온도 등) 영향으로 치수 편차가 발생했습니다.
  - 해결: 모델 치수 재검증 후 스케일을 조정하고, 테스트 출력으로 프린터 설정(유량/온도/속도/수축 보정)을 최적화했으며, 재료 특성을 반영해 출력 방향과 지지대 조건을 조정했습니다.
  - 결과: 조립 공차를 만족하는 출력물을 확보하여 기구 조립을 정상적으로 진행했습니다.

- **카메라 인식 값 ↔ 직교좌표 로봇 전송(통신) 불안정**
  - 문제: 카메라에서 인식한 좌표/오프셋 값을 직교좌표 로봇 제어부로 전달하는 과정에서 값 누락/오류 등으로 정상적인 송신이 어려웠습니다.
  - 원인: 데이터 포맷 불일치, 통신 속도/버퍼 처리, 예외 상황 처리 미흡 등으로 전송 안정성이 떨어졌습니다.
  - 해결: 전송 포맷을 명확히 정의하고, 코드를 단계적으로 수정하면서 로그 기반 디버깅을 수행해 예외 처리와 버퍼 관리를 보완했습니다.
  - 결과: 인식 결과가 안정적으로 송신되며 로봇 제어에 정상 반영되었습니다.

- **파워서플라이 제작 중 손상(과전류/단락 위험)**
  - 문제: 파워서플라이 제작을 위해 제품을 분해·조립하는 과정에서 전원부가 손상되는 위기가 있었습니다.
  - 원인: 배선/결선 실수, 단락 가능성, 보호 회로 부족 등으로 전원부에 과부하가 걸릴 위험이 있었습니다.
  - 해결: 예비 전원부를 준비해 리스크를 낮추고, 결선을 하나씩 재점검하며 보호 요소(배선 정리, 절연, 연결 순서 점검 등)를 강화했습니다.
  - 결과: 시스템 요구에 맞는 안정적인 전원 공급용 파워서플라이를 구성했습니다.

- **타이밍 벨트 장력 저하로 인한 구동 불량**
  - 문제: 3축 직교좌표 로봇 구동 중 타이밍 벨트가 느슨해져 슬립이 발생하고 동작이 불안정해졌습니다.
  - 원인: 장력 부족 및 반복 구동에 따른 장력 변화로 전달 오차가 발생했습니다.
  - 해결: 벨트 장력을 재조정해 타이트하게 맞추고, 구동부 체결 상태를 재점검했습니다.
  - 결과: 축 이동이 안정화되고 반복 동작 신뢰도가 개선되었습니다.

- **장시간 구동 시 스텝 누적 오차(부동소수점 누적)**
  - 문제: 3축 직교좌표 로봇을 장시간 구동하면, 이동 스텝 값을 float(부동소수점) 기반으로 누적 계산하는 과정에서 미세 오차가 계속 쌓여 목표 위치 대비 편차가 발생했습니다.
  - 원인: 부동소수점 연산의 근사 오차가 반복적으로 누적되면서, 스텝/좌표 환산 값이 장기적으로 드리프트하는 특성이 있었습니다.
  - 해결: 누적 오차를 주기적으로 초기화하기 위해 **5분마다 홈 포지션(Home Position)으로 자동 이동**하여 기준점을 재정렬하고, 내부 좌표/스텝 누적 값을 **0으로 리셋**하는 방식으로 운용 로직을 추가했습니다.
  - 결과: 장기 구동에서도 좌표 드리프트가 억제되어 반복 작업 정밀도와 신뢰성이 개선되었습니다.
